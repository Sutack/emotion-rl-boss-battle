# Emotion-Adaptive Boss Battle

파이썬 기반 딥러닝 수업 프로젝트입니다.  
플레이어의 얼굴 감정을 인식하고, 강화학습(PPO)으로 학습된 보스 AI의 난이도를 실시간으로 조절하는 탑다운 2D 아레나 게임입니다.

## 기술 스택
- Python, PyTorch
- Facial Emotion Recognition (FER2013 기반 CNN)
- Reinforcement Learning (PPO, Stable-Baselines3)
- pygame / Streamlit (시각화 및 데모)
